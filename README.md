# ColorOS Box 刷机工具箱

> 专为 ColorOS 及一加/OPPO 设备打造的 Windows 刷机工具箱

## 📱 功能特性

- ✅ 自动识别 45+ 个分区类型
- ✅ 支持 Super 动态分区刷写
- ✅ 基带分区独立刷写
- ✅ 解包 OTA 包并刷写（payload.bin）
- ✅ ADB 投屏（scrcpy 集成）
- ✅ 手机数据备份与恢复
- ✅ 自动检测设备模式
- ✅ 彩色界面 + 进度条

## 🚀 快速开始

1. 下载并解压工具箱
2. 将 ADB、Fastboot 等工具放入 `拓展` 文件夹（可选）
3. 双击运行 `ColorOS Box.exe`
4. 输入密码：`bzdabox123`
5. 选择功能开始刷机

## 📖 使用说明

### 刷机流程
1. 选择功能 1/3/4/5
2. 输入镜像文件夹路径
3. 选择刷写模式（手动/全部/跳过）
4. 自动进入 FastbootD 模式
5. 自动刷写
6. 选择是否重启

### 备份恢复
- 功能14：备份手机数据
- 功能15：恢复手机数据

### 基带刷写
- 功能 B：单独刷写基带分区

## 🔧 工具支持

程序会自动检测以下位置的工具：
- 系统 PATH 环境变量
- `拓展` 文件夹（支持子目录）
